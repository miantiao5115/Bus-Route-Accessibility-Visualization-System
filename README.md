## 简介

TAZ:Traffic Analysis Zone，可以翻译为交通分析区、交通小区。

TAZ-Accessibility-Visualization-System 是一个简单的交通分析区可达性可视化的web应用，通过该应用可以直观的看出各个交通分析区的可达性。
